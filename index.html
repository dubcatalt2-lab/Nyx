Shortcuts

Nyx uses Left Alt for browser controls so the shortcuts still feel familiar while staying inside Nyx.

Switch tabs left to rightAlt + 1-9
Focus the URL barAlt + L / D
Open a new tabAlt + T
Close the current tabAlt + W
Reload the current tabAlt + R
Go back or forwardAlt + Arrows
Select all textAlt + A
Copy, cut, pasteAlt + C/X/V
Undo or redo text editsAlt + Z/Y
Close menus and suggestionsEsc
On ChromeOS only, pressing Alt + Tab dims Nyx like brightness 0. Your hardware brightness cannot be changed by a website, so Nyx uses a black privacy overlay instead.
